# Geo Project
## Cloning the project
```
git clone https://github.com/eillaste/geo-project.git
```
```
cd geo-project
```
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Go to:
http://localhost:8080/geo

### Also available online:
https://www.trialda.com/geo/
